# Электронный журнал

Веб-приложение для учёта посещаемости и оценок студентов. Поддерживает две роли: **Студент** и **Преподаватель**.

## Технологии

| Часть | Стек |
|---|---|
| Frontend | React 19 + Vite + TypeScript + Ant Design |
| Backend | Node.js + TypeScript + Express |
| База данных | SQLite (better-sqlite3 + Drizzle ORM) |
| Аутентификация | JWT |
| Файлы | Multer (локальное хранилище) |

## Запуск проекта

### 1. Клонировать репозиторий

```bash
git clone https://github.com/Alten-by/Journal_kbp.git
cd Journal_kbp
```

### 2. Запустить бэкенд

```bash
cd backend
npm install
npm run db:migrate    # создать таблицы в БД
npm run db:seed       # заполнить тестовыми данными
npm run dev           # сервер запустится на http://localhost:3000
```

### 3. Запустить фронтенд

В новом терминале:

```bash
cd frontend
npm install
cp .env.example .env  # скопировать переменные окружения
npm run dev           # откроется на http://localhost:5173
```

## Функциональность

### Студент
- Расписание на неделю
- Журнал оценок и посещаемости по всем предметам
- Страница предмета с разбивкой по типам работ (лабы, практики, контрольные, тесты)
- Страница лабораторной: просмотр задания, сдача файла, статус проверки и комментарий преподавателя

### Преподаватель
- Расписание на неделю
- Журнал группы: добавление уроков, выставление оценок и посещаемости
  - **ЛКМ** — ввод оценки (1–10)
  - **ПКМ** — пропуск (Н)
  - **СКМ** — опоздание: если урок идёт сейчас — минуты считаются автоматически; если урок уже закончился — выбор: ввести минуты или поставить «О»
- Программа предмета: создание и редактирование лабораторных, загрузка ТЗ, настройка команд
- Экран сдачи лаб: просмотр сданных работ, выставление оценки и комментария

### Дополнительно
- Адаптивный интерфейс (мобильные устройства)
- Подсветка строки/колонки при наведении в журнале
- Выделение новых и отчисленных студентов цветом
